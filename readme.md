> 小组：转生成为异世界贤者之魔王竟是我的队友

## 仓库说明

本仓库为**2022-2023春季秋期软件工程**课程新闻搜索系统项目主仓库，主要存储各项目子模块 & 项目文档

## 技术选型

- 网页前端：Vue
- 网页后端：Django
- 数据库：PostgreSQL
- 爬虫：Scrapy
- 检索后端：Elasticsearch
